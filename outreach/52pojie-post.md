## 发帖注意事项

- **截图必须有**：附上 README 截图（展示状态标记）和 zhuiju.me 网站截图
- **排版用分隔线**：`━━━━` 或 `----` 分割各部分，吾爱用户扫帖习惯
- **"开源"要突出**：吾爱用户信任开源项目，认为不容易卷跑路
- **主动 @版主**：发完后可以 @ 版区版主申请加精，格式规范的帖子通过率高
- **及时回复**：吾爱用户会问"有没有更多配置地址"、"XXX 能用吗"，及时回复维持热度
- **不要发多余链接**：帖子里只放 zhuiju.me 和 GitHub 两个链接，Gitee 可以在评论补充
