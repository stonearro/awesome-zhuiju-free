### 推荐理由

找追剧资源最烦的事不是"没有资源"，而是"不知道哪个资源今天还能用"。
收藏夹里存了一堆网站，打开十个有八个挂了，剩下两个全是广告弹窗。

Awesome Zhuiju Free 解决的就是这个问题：

- 人工筛选，只收录免费、无广告的资源
- GitHub Actions 每天早上自动检测每个资源能否访问，结果标在页面上
- 完全开源，数据公开，没有黑箱

### 收录分类

- 在线影视（23个）
- 网盘搜索、磁力BT（各4个）
- 字幕资源
- TVBox / 影视仓配置地址（14个）
- IPTV 订阅源
- 影视APP
- 开源播放器项目

目前共 62 个资源，持续更新。

### 使用方式

不需要注册、不需要安装，直接打开网页或 GitHub 页面就能用：

- 网站：https://zhuiju.me
- GitHub：https://github.com/laoma2053/awesome-zhuiju-free

### 状态说明

每个资源旁边有可用状态标注：🟢 可访问 / 🟡 访问受限 / 🔴 无法访问。
检测由 GitHub Actions 每天北京时间09:00左右自动运行，不依赖人工维护。

如果你发现某个资源失效，或者有好用的免费无广告资源想推荐，直接在 GitHub 提 Issue 就可以。
